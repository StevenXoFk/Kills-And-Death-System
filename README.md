# Kills-And-Death-System
This plugin adds a simple but functional kills and kills system for Minecraft Bedrock servers with Endstone.
Saves the statistics of each player in a JSON file and allows them to be consulted using the command:

``/stats [player]``

También muestra mensajes personalizados de muerte en el chat, indicando quién mató a quién y cuántas kills tiene cada jugador.
